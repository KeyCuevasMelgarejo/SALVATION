# Salvation - Descripción del Proyecto
**El proyecto cuenta con los siguientes elementos:**
- Sistema de particulas para el efecto lluvia
- Stellar Environment
- RigidBodies
- Collisions
- Movements
- Animations
- Dancing
- Shots
- Jummping
- Running
- Attack
- Sound
- Background Music
- Average damage
- Lose / Win
- Collect
- Missions
